## 4) If Screen Becomes White / Blank

### English

If the page shows blank white screen:

1. Close the tab fully.
2. Open the link again in a **new tab**.
3. If needed, open in **Incognito / Private mode**.
4. Refresh once after internet is stable.
5. If still not loading, clear browser cache for the site and reopen.

### हिंदी

अगर page पर सफेद / blank screen दिखे:

1. Tab को पूरी तरह बंद करें।
2. लिंक को **नई tab** में फिर से खोलें।
3. जरूरत हो तो **Incognito / Private mode** में खोलें।
4. Internet stable होने पर एक बार refresh करें।
5. फिर भी issue हो तो browser cache clear करके दोबारा खोलें।

## 6) Simple Role Understanding

### English

- **Super Admin**: manages the whole system
- **Vibhag Pramukh**: sees higher-level overview and approvals
- **Aayam Pramukh**: reviews work for their Aayam
- **Unit Head**: creates and manages unit-level work
- **Karyakarta**: contributes work with limited access

### हिंदी

- **Super Admin**: पूरे सिस्टम को manage करता है
- **Vibhag Pramukh**: overview और approvals देखता है
- **Aayam Pramukh**: अपने Aayam का काम review करता है
- **Unit Head**: unit level का काम create और manage करता है
- **Karyakarta**: limited access के साथ काम contribute करता है
